== Description ==
This plugin adds a strict AND search behavior for WordPress search queries and includes a widget/shortcode frontend form.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/faranux-strict-search` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the widget or use the shortcode `[faranux_strict_search]`.
